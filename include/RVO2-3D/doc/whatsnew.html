What Is New in RVO2-3D Library

Three Dimensions

In contrast to RVO2 Library, RVO2-3D Library operates in 3D workspaces. It uses a three dimensional implementation of Optimal Reciprocal Collision Avoidance (ORCA) for local collision avoidance. RVO2-3D Library does not replace RVO2 Library; for 2D applications, RVO2 Library should be used.

Structure of RVO2-3D Library

The structure of RVO2-3D Library is similar to that of RVO2 Library. Users familiar with RVO2 Library should find little trouble in using RVO2-3D Library. RVO2-3D Library currently does not support static obstacles.
